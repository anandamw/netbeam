**Netbeam** adalah aplikasi desktop berkecepatan tinggi yang memungkinkan Anda untuk mentransfer file berukuran raksasa antar komputer dalam satu jaringan lokal (Wi-Fi / LAN) secara instan, tanpa memerlukan koneksi internet, tanpa kabel, dan tanpa perantara server luar. 

Dirancang dengan antarmuka yang sangat mudah dipahami sehingga siapa saja dapat menggunakannya.

## Get the app

- [Windows](https://github.com/anandamw/netbeam/releases/latest) (read [how to run](#usage-examples))
- [Android](https://github.com/anandamw/netbeam/releases/latest)

## Must-know tips

- **Super Cepat** memanfaatkan batas maksimal kecepatan `router` Wi-Fi atau kabel LAN Anda.
- **Network Radar** tidak perlu pusing mencari IP Address. Fitur radar otomatis menemukan PC teman Anda.
- <kbd>Accept</kbd> atau <kbd>Reject</kbd> triggers incoming file permissions. Anda punya kendali penuh atas keamanan privasi Anda.
- <kbd>Drag</kbd> + <kbd>Drop</kbd> triggers multi-file upload langsung dari *File Explorer* Anda.
- There are many other features like **Built-in Chat** dan **Transfer Memory** untuk menyimpan "Jalan Pintas".

## Usage examples

Pastikan Anda dan penerima file berada di jaringan Wi-Fi / LAN yang sama.

- Nyalakan radar dan hubungkan ke perangkat yang dituju:

  ```text
  1. Klik tombol "Start" pada panel My Station
  2. Perhatikan panel Radar untuk melihat nama PC tujuan
  3. Klik nama PC tersebut dan tekan "Connect"
  ```

- Mengirim file (Pihak Pengirim):

  ```text
  Seret file / folder dari Desktop Anda
  Lepaskan (Drop) tepat ke dalam jendela aplikasi Netbeam
  ```

- Menerima file (Pihak Penerima):

  ```text
  Penerima akan mendapatkan pop-up konfirmasi
  Klik tombol "Accept" dan file akan langsung meluncur dengan kecepatan penuh!
  ```
