# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-17

### Added
- **Lightning Fast P2P Transfer**: Core TCP/IP implementation for extremely fast local network file transfers.
- **Folder Transfer Support**: Send entire directories seamlessly while preserving the original structure.
- **Auto-Discovery (Radar)**: Local network UDP broadcast to automatically detect active Netbeam instances.
- **Smart Queue & Real-time Progress**: Visual feedback for file sending with per-chunk progress updates.
- **Device Pairing & Security**: Bluetooth-style 6-digit PIN verification to authenticate unknown connections.
- **Trusted Devices List**: Save paired IP addresses to allow one-click file transfer for known devices.
- **Transfer History**: Keep track of successful and failed file sends/receives.
- **Search & Filter**: Find past transfers instantly using file name or IP address.
- **Clean UI**: Modern, dark-mode, glassmorphism-inspired design with fully responsive components.
